# About

This is a complete remake of our original project <a href="https://github.com/FefanAM/Rib.it">located here.</a>

It is inspired by the game <a href="https://store.steampowered.com/app/1061090/Jump_King/">`Jump King`</a>.

### Control

- You ascend the obstacles by jumping from one to the next.

- To jump, hold <kbd>SPACE</kbd> to charge jump intensity and aim using <kbd>A</kbd> ,<kbd>D</kbd>.

- Let go of space to jump.

# How to use

To run the webpage, open `main.html` in your browser.

To run the game you will need to install python and pygame.

### Installing python

Download python from <a href="https://apps.microsoft.com/detail/9pnrbtzxmb4z?">Microsoft Store</a> or <a href="https://www.python.org/">official python website.</a>

### Installing pygame

To install pygame, open command prompt and type or copy:

    py -m pip install pygame


> [!WARNING]
> If your computer runs Windows on aarch64 (arm64) architecture,
> You must install the x86 version of python because pygame does not have an aarch64 version for Windows.
