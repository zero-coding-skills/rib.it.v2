# Rib.it v2

This is a complete remake of our original project <a href="https://github.com/FefanAM/Rib.it">located here.</a>

It is inspired by the game 'Jump King'.

# Download

Download the repository or download <a href="https://github.com/zero-coding-skills/rib.it.v2/releases">here</a>.

# How to use

To run the webpage, open `main.html` in your browser.

To run the game you will need to install python and pygame.

### Installing python

Download python from <a href="https://apps.microsoft.com/detail/9pnrbtzxmb4z?">Microsoft Store</a> or <a href="https://www.python.org/">official python website.</a>

### Installing pygame

To install pygame, open command prompt and type or copy:

    py -m pip install pygame
